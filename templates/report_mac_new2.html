6sense Host Audit

This tool compares 6sense hosts available in MDM vs Crowdstrike. Upload CSV files from different sources to identify discrepancies.


How does this work?

  • Download Host data as CSV from different MDM sources.
    • From Jamf, Intune, Jumpcloud
  • Download host data as CSV from Crowdstrike.
    • Filter OS: Mac
    • Filter OS: Linux
    • Filter OS: Windows
  • Input all the files to create complete report.
